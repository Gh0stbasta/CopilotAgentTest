Todo App - REST API

Welcome to the Todo App! The REST API is now available.

Available API Endpoints:

GET /api/todos
Retrieve all todos
POST /api/todos
Create a new todo
Body: { "title": "Task name", "completed": false }
PUT /api/todos/:id
Update an existing todo
Body: { "title": "Updated task", "completed": true }
DELETE /api/todos/:id
Delete a todo

Test the API using:

  • Postman
  • cURL commands
  • Browser developer tools

Data is stored in todos.db SQLite database.
